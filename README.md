## 🚩 진행 방식

- `1주일` 동안 `정한 분량`을 읽는다.
- 읽은 각 장은 `.md` 파일로 만들어서 문서화 진행
- 1주 일 동안 읽은 `감상평`을 담아서 PR을 매 주 올림
- 매 주 `수요일`에 `.md`파일을 보면서 발표 진행하며, 의견 나누기
- 발표자는 각 목차에 따라서 진행하고, 본인이 느낌점에 대해 이야기 하고, 중간중간 의견이 있거나, 하고 싶은 말이 있는 사람은 이야기를 하는 방향으로 진행
